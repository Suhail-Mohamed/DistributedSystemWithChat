How to work

  1. Read the lesson HTML.
  2. Run the corresponding lab.
  3. Open the single lesson question file under questions/.
  4. Write answers in a separate Markdown file in the lab folder.
  5. Commit the answer file and ask for review in chat.

New lessons keep all prompts in one Markdown file. The HTML pages contain teaching material and lab instructions only.
